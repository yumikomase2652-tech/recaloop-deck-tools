# RecaLoop Deck Tools

RecaLoop Deck Tools is an open-source toolkit for structuring educational materials into machine-readable deck data.

It helps tutors, students, and EdTech developers convert textbook-like content into hierarchical formats such as:

- material
- chapter
- section
- item / problem

The goal is to make educational materials easier to reuse for review scheduling, assignment generation, and learning analytics.

## Why this matters

Many tutors and learners manage textbook progress manually. However, textbook structures are often difficult to connect with digital learning tools.

This project provides a simple foundation for converting educational content into structured data that can be used in spaced repetition, assignments, and learning management systems.

## Planned features

- CSV to JSON conversion
- Deck structure validation
- Example educational deck files
- TypeScript package
- CLI support
